Software Feature

 
The major feature provided by the system is registration , Booking of court/turf and Payment gateway


4.1 Registration

4.1.1 Description and Priority
        This feature has the highest priority as it is one of the main purpose of the system.
4.1.2 Stimulus/Response Sequences
         Customers have to register to the page. After registraion customer will get an userid and password. This userid and password will be usefull to the future booking. As the data of that customers is already been stored in the database so that it will be linked to the requested result and remaining things will be done by the system.
4.1.3 Functional Requirements
         Functional requirement for the system is that users’ data  must be store in the databaseand he/she is allowed by the admins to access the functionalities.









4.2 Booking of court/turf
4.1.1 Description and Priority
        This feature has the highest priority as it is one of the main purpose of the system.
4.1.2 Stimulus/Response Sequences
         Customers have to login to the page. After login , customer will get an information about availabilty of the court/turf  and on what time. Customer also get an information, which activities is going to happens & about the  fees Structure of the court/ground .
4.1.3 Functional Requirements
         Functional requirement of the system is that, the information available for the court/turf availabilty, total fare struture of the court/turf and time in the database will be fetch as per the customer requirement.


4.3 Payment gateway

4.1.1 Description and Priority
        This feature has the less priority as we compare with the above two option.
4.1.2 Stimulus/Response Sequences
         After booking the court/ turf  on a particulare date and time , customer will redirected to the payment gateways page. Customer can pay a payment via CreditCard, Debitcard to the admin account.Cash on delivery option is also avaialble.
4.1.3 Functional Requirements
         Functional requirement for the sytem is that customer will get generated payment bill and get an id card of the court/turf.
